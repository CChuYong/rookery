# v0.4.0 Release Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish rookery v0.4.0 from the current `origin/main` with reviewed release notes and complete desktop artifacts.

**Architecture:** The release tag is the version source of truth. Source package versions remain unchanged because `.github/workflows/release.yml` stamps packages from `v0.4.0` while building. GitHub Actions validates the root and desktop workspaces, builds each platform, and attaches artifacts to the GitHub Release.

**Tech Stack:** Git, GitHub CLI, GitHub Actions, Node.js 22, npm, Vitest, TypeScript, Electron Builder
